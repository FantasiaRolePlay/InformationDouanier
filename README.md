## Commandes Douaniers.

- Whitelist Ouverte.
```shell
+wlon
```

- Whitelist Fermée.
```shell
+wloff
```

- Ajouté un joueur dans la whitelist.
```shell
+addwluser @joueurs
```

- Supprimé un joueur dans la whitelist.
```shell
+removewluser @joueurs
```
